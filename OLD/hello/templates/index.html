Welcome to ACCrobat!

With ACCrobat, you can analyze and visualize TCGA .BED files easily for fast insights into tumor genetic data.

Get Started! Source on GitHub

How this app works

  • This app was developed in Python and deployed to Heroku using GitHub as the repository.
  • ACCrobat takes TCGA .BED files and summarizes their data by pivoting and visualizing the information.
  • The output presented by this application can help us to understand the genetic differences of tumors from looking at various cell samples.
